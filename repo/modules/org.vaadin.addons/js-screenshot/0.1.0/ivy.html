Description

This is a simple widget to take client-side screenshots using html2canvas.js (http://html2canvas.hertzen.com/).

Be aware that due to the same origin policy external/svg images are not supported.

Features:

  1. take client-side screenshots (w/o the need for an applet)
  2. transfers screenshot to server application
  3. select root widget for the screenshot creation

Requirements:

  1. jQuery 1.6+ (required to be included before widgetset)
