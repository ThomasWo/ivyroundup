Description

The Outline module provides two Swing components: ETable which is a JTable extension with extra features like column sorting and row filtering. Outline which is a tree-table like component capable of showing a tree hierarchy of nodes in a table.

This module has no dependencies on the rest of the NetBeans modules and can be used as a standalone jar.
