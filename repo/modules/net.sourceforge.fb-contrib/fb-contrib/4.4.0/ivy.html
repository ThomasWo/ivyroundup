Description

fb-contrib™ is an extra detector plugin to be used with the static bug finder FindBugs™ (findbugs.sourceforge.net). Just download the fb-contrib.jar file, and place it in your FindBugs™' plugin directory. FindBugs™ will automatically pick up the jar file, and incorporate these detectors with its own.
