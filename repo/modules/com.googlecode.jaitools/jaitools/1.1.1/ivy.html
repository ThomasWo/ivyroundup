Description

Additional operators, utility classes and the Jiffle scripting language to work with the Java Advanced Imaging library

Public Configurations

Name Description Extends
all Everything
default Alias for 'all' conf
attributeop An OpImage class that derives one or more attributes from a source image. Source image pixels are passed through to the destination image unchanged (similar to JAI's StatisticsOpImage).
contour Contour operator attributeop,utils
kernelstats KernelStats operator utils
maskedconvolve Adds selectivity to the JAI convolve operation. An ROI canbe used to control which source image pixels are convolved and/orwhich neighbourhood pixels are included in the convolution.
rangelookup A Lookup operation that can handle both integral and continuous data types utils
regionalize Identifies (sufficiently) uniform regions in the source image, allocateseach a unique integer ID, and generates an output image with these IDsas pixel values utils
utils Support and utility classes used by other JAI-tools components and available for general use.
vectorbinarize Creates a binary image from an input JTS Polygon or MultiPolygon utils
vectorize Vectorizes the boundaries of regions with uniform value attributeop,utils
zonalstats Calculates summary statistics, optionally within zones defined bya zone image, for values in a data image utils
