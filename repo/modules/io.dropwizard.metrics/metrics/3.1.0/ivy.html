Description

Metrics is a Java library which gives you unparalleled insight into what your code does in production. Metrics provides a powerful toolkit of ways to measure the behavior of critical components in your production environment.

Public Configurations

Name Description Extends
annotation A dependency-less package of just the annotations used by other Metrics modules.
core Metrics Core
ehcache An Ehcache wrapper providing Metrics instrumentation of caches. core
ganglia A reporter for Metrics which announces measurements to a Graphite server. core
graphite A reporter for Metrics which announces measurements to a Ganglia cluster. core
healthchecks An addition to Metrics which provides the ability to run application-specific health checks, allowing you to check your application's heath in production. jvm
httpasyncclient An Apache HttpAsyncClient wrapper providing Metrics instrumentation of connection pools, request durations and rates, and other useful information. httpclient
httpclient An Apache HttpClient wrapper providing Metrics instrumentation of connection pools, request durations and rates, and other useful information. core
jdbi A JDBI wrapper providing Metrics instrumentation of query durations and rates. core
jersey A set of class providing Metrics integration for Jersey, the reference JAX-RS implementation. This module is for the old Jersey 1.x core,annotation
jersey2 A set of class providing Metrics integration for Jersey, the reference JAX-RS implementation. core,annotation
jetty8 A set of extensions for Jetty 8 which provide instrumentation of thread pools, connector metrics, and application latency and utilization. core
jetty9 A set of extensions for Jetty 9.1 and higher which provide instrumentation of thread pools, connector metrics, and application latency and utilization. core
jetty9-legacy A set of extensions for Jetty 9.0 which provide instrumentation of thread pools, connector metrics, and application latency and utilization. core
json A set of Jackson modules which provide serializers for most Metrics classes. healthchecks
jvm A set of classes which allow you to monitor critical aspects of your Java Virtual Machine using Metrics. core
log4j An instrumented appender for Log4j 1.x. core
log4j2 An instrumented appender for Log4j 2.x. core
logback An instrumented appender for Logback. core
servlet An instrumented filter for servlet environments. core
servlets A set of utility servlets for Metrics, allowing you to expose valuable information about your production environment. jvm,json,jetty9
default Alias for core configuration core
