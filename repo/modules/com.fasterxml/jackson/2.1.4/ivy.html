Description

Inspired by the quality and variety of XML tooling available for the Java platform (StAX, JAXB, etc.), the Jackson is a multi-purpose Java library for processing JSON data format. Jackson aims to be the best possible combination of fast, correct, lightweight, and ergonomic for developers.

Public Configurations

Name Description Extends
core Contains streaming parser, generator
annotations for annotation-based configuration core
databind for data binding, tree model core
module-jaxb-annotations Support for using JAXB annotations as an alternative to native Jackson annotations, for configuring data binding. databind
jaxrs-json-provider Functionality to handle JSON input/output for JAX-RS implementations (like Jersey and RESTeasy) using standard Jackson data binding. databind
default Most common configuration of core, databind, and annotations. databind,annotations
