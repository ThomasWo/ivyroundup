Description

Inspired by the quality and variety of XML tooling available for the Java platform (StAX, JAXB, etc.), Jackson is a multi-purpose Java library for processing JSON data format. Jackson aims to be the best possible combination of fast, correct, lightweight, and ergonomic for developers.

datatype-jsr310 Add-on module to support JSR-310 (Java 8 Date & Time API) data types. databind
datatype-jsr353 Add-on module for Jackson (http://jackson.codehaus.org) to support working with JSR-353 (JDK Json-processing API) node typesm via data-binding databind
default Most common configuration of core, databind, and annotations. databind
jaxrs-base Pile of code that is shared by all Jackson-based JAX-RS providers. databind

jaxrs-xml-provider Functionality to handle XML input/output for JAX-RS implementations (like Jersey and RESTeasy) using standard Jackson data binding. jaxrs-base,module-jaxb-annotations,dataformat-xml
jr-objects Simple data-binding that builds directly on 'jackson-core' (streaming), has no other dependencies, and provides additional builder-style content generator
module-afterburner Jackson (https://github.com/FasterXML/jackson) extension module used to enhance access functionality using bytecode generation. databind
module-guice Stuff to make integration with Guice a bit easier databind
module-jaxb-annotations Support for using JAXB annotations as an alternative to 'native' Jackson annotations, for configuring data binding. databind
