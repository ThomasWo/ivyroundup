Description

GATE (General Architecture for Text Engineering) is the premire Java natural language processing suite.

Note that CREOLE plugins (artifacts with Ivy type of "creole") are packaged as zip files due to Ivy limitations, and need to be unzipped into an appropriate plugins directory in order to be used.
