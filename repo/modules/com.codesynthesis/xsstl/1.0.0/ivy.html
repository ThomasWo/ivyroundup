Description

XML Schema Standard Type Library (XSSTL) is a collection of universally-useful data types defined in the W3C XML Schema language.

*** Note *** This module uses restricted build instructions; You must configure your packager resolver with restricted="false". See the Packager Resolver Documentation for more information.
